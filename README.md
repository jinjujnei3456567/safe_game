# 安全主题互动小游戏 🛡️
这是一个面向中小学生的国家安全主题互动教育小游戏，通过剧情闯关、答题互动的形式，普及国家安全相关知识。

## 项目介绍
- 项目名称：安全卫士
- 开发目的：提升青少年国家安全意识，普及《国家安全法》相关知识
- 技术栈：HTML + 原生JS + 视频资源
- 部署地址：https://jinjunei3456567.github.io/safe_game/

## 功能说明
- 剧情闯关模式：通过视频引导+选择题互动推进剧情
- 知识点覆盖：总体国家安全观、网络安全、校园安全等
- 适配场景：校园安全教育、线上科普活动

## 本地运行
直接打开 `index.html` 即可在浏览器中运行，无需额外依赖。
